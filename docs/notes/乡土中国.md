# 乡土中国

---

- **第一章**：中国基层乡土社会是熟人社会
  
  - 农民靠土地谋生
  
  - 人口粘在土地上不流动
  
  - 现实因素决定人们聚村而居
    
    - 内部熟悉
      
      - 人与人：从心所欲不逾矩，在一起生长而发生的礼俗社会（情感占优）并非为了任务而结合的法理社会（理性占优）
      
      - 人与环境：熟悉
    
    - 村落之间孤立隔阂

- **第二至三章**：空间不流动的亲密性和时间重复的单调性带来生活定型
  
  - 人需要文字积累经验打破时空束缚，乡土社会生活定型不需要额外经验
  - 文字不如语言，语言不如特殊语言（声音）
  - 如果乡土社会不现代化、基层不变化，文字没有必要下乡

- **第四至七章**：乡土社会人际关系结构是*差序格局*
  
  - 维系定型生活的是差序格局（中西方各有偏胜）
    
    - 中国乡土社会的结构是同心圆（界限伸缩）：差序格局-以自己为中心-乡土风格自食其力强调克己和修身、孝悌忠信是私人关系
    
    - 西方社会是捆柴（界限清晰）：团体格局-人人平等、权利义务分明-游牧风格相互依赖，强调遵纪守法
  
  - 公私不分来源于所站位置不同
  
  - 差序格局的纽带人伦来自儒家
  
  - 差序格局导致大家庭（结构上而并非人口上）->小家族（重视纵向关系而并非横向夫妻关系）
    
    - 男女有别（生活隔离、心理疏远）便于抑制感情促进社会稳定
    
    - 感情的激动会改变原有的社会关系、感情的淡漠能稳定社会关系

- **第八至九章**：乡土社会是礼治社会
  
  - "人治"也需要规则，“法治”也需要人
  - 礼治
    - 中性词、不是慈善和文明而是社会公认的规范
    - 维持的力量是传统*礼治的根本区别*
    - 不是通过外界力量强制施加而是通过教化让人主动服从
      - 乡土社会反对诉讼、鼓励教化（长老调解）
      - 不知道礼仪秩序是耻辱、不知道法律没关系

- **第十至十一章**：想吐社会的权力是无为的、教化的
  
  - 社会冲突-横暴权力-阶级压迫获得利益-农村经济剩余太少不能维持
  
  - 社会合作-同意权力-社会契约分工合作-小农经济自给自足不能产生
  
  - 乡土社会的权力结构是教化性的（类似旅客须知、家有严父族有长老）
    
    - 教化不是剥削甚至损己利人
    
    - 教化没有征得被教化者的同意
    
    - 稳定的文化滋养了教化新成员的权力
    
    - 变迁的社会发展代替了经验、弱化了教化

- **第十二至十四章**：社会权力的变迁
  
  - 社会变化速率决定血缘作用强弱
    
    - 血缘社会稳定、血缘地缘合一、迁移也依旧带着血缘
    
    - 社会的发展、地缘的流动带来了从乡土到现代的变迁
  
  - 乡土社会并非静止而是缓慢变化
    
    - 慢的长老权力
    
    - 快的时事权力
    
    - 社会结构不是自身变动
    
    - 不适应发展就会有文化英雄
      
      - 长老权力不容反对和质疑
      
      - 交替过程中就会名实分离
    
    - 长老社会靠欲望行事
      
      - 欲望合乎人类生存条件
      
      - 欲望的目的和结果往往分化
      
      - 欲望->行为->经验->文化
    
    - 现代社会靠计划行事
      
      - 根据科学制定计划满足需要
      
      - 知识即权力
